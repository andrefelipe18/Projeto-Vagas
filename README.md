# vagas-front 

## É possivel nele criar e visualizar vagas

## Como rodar o projeto

### Pré-requisitos

- Node.js
- NPM ou Yarn

### Instalação
#### Clonando o projeto
```
git clone

```
#### Instalando as dependências
```
npm install
```
ou
```
yarn install
```
#### Rodando o projeto
```
npm start
```
ou
```
yarn start
```
